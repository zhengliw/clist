void * clist_append ( struct clist **  clist)

Append an item.

Append an item at the end of the list while still staying on the current index.

Parameters
clistThe list to be operated on. This has to be a pointer pointing to the list pointer, since we are modifying clist::end.
Returns
A pointer to the item on the newly created node. If failed appending a node at all, returns NULL.

struct clist * clist_init ( size_t  item_size)

Create a clist.

This function is used to create a clist, performing all necessary steps while doing this.

Parameters
item_sizeSpecifies how big one "item" is. An item is a block of data that is defined by you and will be stored on the linked list. No explicit type for an item is required, Just provide the size of the item.
Returns
a pointer to the clist that is created, NULL if failed.
